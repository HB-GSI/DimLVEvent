bin:			LV-Dim-Interface files for specific LV and DIM version, used by CS, CS++ or in standalone LV-projects

DIMIndicators:		LV-library for indicators which connect automaticaly to DIM

LVDimInterface:		LV-library for the LV-Dim_Interface. This library has to be added to a LV project in order to use DIM

NodeMon:		C-library and dlls for NodeMon as well as the C sourcen

NodeMon-LV:		LV-library for NodeMon

Sourcen-DimLVInterface:	All C sources needed to compile the LV-Dim-Interface
